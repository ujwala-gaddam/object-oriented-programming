# object-oriented-programming
Java Object-Oriented Programming concepts with examples, exercises, and real-world implementations.


This repository contains Java Object-Oriented Programming (OOP) concepts and hands-on examples.

Topics Covered:
- Classes and Objects
- Constructors
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Interfaces
- Method Overloading & Overriding
- Association, Aggregation, and Composition
- Exception Handling
- Collections Framework
- SOLID Principles
- Real-world OOP Projects
